# Cementos Progreso Ambiente

Aplicación web para gestión ambiental, reportes y entrada de datos de Cementos Progreso.

## Stack

- React 19 + TypeScript
- Vite
- React Router
- Lucide Icons

## Desarrollo

```bash
npm install
npm run dev
```
